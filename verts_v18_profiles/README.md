# Access Control (User Profiles) - Odoo 18

This module provides a new way to manage user rights through functional profiles.

## Features
- Create user profiles as fictive users
- Associate profiles to users
- Manage groups through profiles
- Configure which fields to update for linked users
- Track user history

## Installation
1. Copy the `verts_v18_profiles` folder to your Odoo 18 addons path
2. Update the apps list
3. Install the "Access Control" module

## Usage
1. Go to Settings > Users & Companies > User Profiles
2. Create a new profile and configure groups
3. Assign the profile to users

## Version
Upgraded from Odoo 15 to Odoo 18
